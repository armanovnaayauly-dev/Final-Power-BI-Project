# Power BI Dashboard Project

## 📌 Overview
This project presents an interactive Power BI dashboard for analyzing business performance and key metrics.

## 🎯 Objectives
- Build interactive dashboards
- Analyze KPIs and trends
- Support data-driven decision making

## 🧱 Key Features
- Data modeling and relationships
- DAX measures for KPIs
- Interactive filters and slicers
- Visual storytelling with charts

## 📊 Key Metrics
- Revenue
- Sales growth
- Customer activity
- Performance by category/region

## 🛠️ Tools
- Power BI
- Power Query
- DAX

- `/Power BI Final Project.pbix` – Power BI report

## 📈 Outcome
This project demonstrates strong skills in business intelligence and dashboard development.
